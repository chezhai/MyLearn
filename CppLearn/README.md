## 常量指针和指针常量
> 常量指针(const int\*): 该指针指向的那个变量是不可被修改的；但它本身的指向可以被修改。</br>
> 指针常量(int \*const 或者 int const\*): 该指针指向的变量是可以被修改的；但它本身的指向是不能被修改的。</br>
